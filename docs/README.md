# Advanced usage

These guides go deeper than the project [README](../README.md). Reach for them
when you are customizing agent behavior or wiring `diamond-dev` into automation.

- [Custom prompts](custom-prompts.md) — replace the built-in task instructions
  for any phase, and understand the required-context wrapper you cannot override.
- [Automation & CI integration](automation-and-ci.md) — run reports, exit codes,
  run statuses, and notification webhooks for unattended and pipeline use.
- [Observability](observability.md) — logging configuration, the JSONL log
  schema, structured phase fields, and OpenTelemetry trace enrichment.

The [README](../README.md) remains the source of truth for installation,
configuration tables, the workflow overview, two-commit mode, auto-resume, and
security. These guides assume you have read it.
